Detailed Description

This class is a file input stream for reading MEI files.

It uses the libmei C++ library. Under development.

virtual void vrv::MeiInput::SetAppXPathQueries ( std::vector< std::string > &  xPathQueries)
inlinevirtual

Set the xPath queries for selecting specific <rdg>.

By default, the first <lem> or <rdg> is loaded. If one (or more) query is provided, the element matching the specified xPath query will be selected (if any, otherwise the first one will be used).
