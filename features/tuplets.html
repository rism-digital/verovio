Tuplet

Verovio supports tuplets, with no particular constraint on the content (e.g, notes, beams, etc.).

The bracket and the number of the tuplets can be hidden with @bracket.visible="false" and bracket.visible="false" respectively.

MEI tupletSpan elements

Verovio also supports tuplet spans, an alternate MEI approach for encoding tuplets. The limitation of Verovio <tupletSpan> element support is that they are supported as load-only elements. It means that they are converted to <tuplet> elements when the document is loaded, and this is possible only if all the elements contained by the tuplet have the same parent.
