Verovio source code is available on GitHub at https://github.com/rism-ch/verovio

Wiki

The GitHub repository also includes a Wiki with specific documentation https://github.com/rism-ch/verovio/wiki

JavaScript toolkit

The GitHub repository also provides a compiled version of the JavaScript toolkit for each release at https://github.com/rism-ch/verovio/releases

Instructions for using the JavaScript toolkit are given on the toolkit page.

The latest release is always available from:

{% highlight bash %}http://www.verovio.org/javascript/latest/verovio-toolkit.js{% endhighlight %}

The latest development version is available from:

{% highlight bash %}http://www.verovio.org/javascript/develop/verovio-toolkit.js{% endhighlight %}

Previous releases are available from their corresponding directory, e.g.:

{% highlight bash %}http://www.verovio.org/javascript/0.9.9/verovio-toolkit.js{% endhighlight %}
